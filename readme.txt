This plugin creates daily posts with gallery of latest Instagram photos.

== Description ==

This plugin downloads your latest photos from Instagram and creates post with gallery.


== Installation ==

1. Login to admin panel of your WP blog and go to `Plugins` â†’ `Add New`
2. Enter `wp-instagram-digest` and click on Search button
3. Click on link Install bellow plugin name `WP Instagram Digest`
4. Activate plugin by clicking on link `Activate`
5. Register new application for your website on http://instagram.com/developer/
6. Proceed with authentication steps on plugin settings page
